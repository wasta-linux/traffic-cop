# NAME
Traffic Cop - Manage bandwidth usage by app or process.

# SYNOPSIS
**traffic-cop** [OPTION...]

# OPTIONS
**-h**, **--help**
: Show help.

**-d**, **--debug**
: Set log level to DEBUG.

**-V**, **--version**
: Print traffic-cop version number and exit.
